方法 1: gdown（推荐）

# 安装
pip install gdown

# 下载整个文件夹
gdown --folder https://drive.google.com/drive/folders/1DtoxMVO9_Y_nDs2peZtx3pw-U2qYgpd3 -O ./dataset
方法 2: wget/curl（需要分享链接）

# 如果有直接下载链接
wget "https://drive.google.com/uc?export=download&id=FILE_ID" -O output.zip
方法 3: rclone

# 配置 Google Drive
rclone config

# 下载
rclone copy gdrive:Full-Duplex-Bench ./dataset
方法 4: 直接用浏览器下载后拷贝
如果上述方法都不行，可以在 Windows/Mac 下载后通过 scp 拷贝到 Linux：


# 从 Windows 拷贝到 Linux
scp -r ./dataset user@linux-server:/path/to/health-assistant/backend/livekit_agent/test/Full-Duplex-Bench/dataset/
推荐用 gdown，在 Linux 上最简单。
